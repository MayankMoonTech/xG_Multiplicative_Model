 xG Model — Multiplicative Approach

 About the Project
This project builds a simple **Expected Goals (xG)** model using a **multiplicative approach**.

The model predicts how likely a shot is to become a goal (value between 0.0 and 1.0).


 What is xG?
xG (Expected Goals) tells us the **chance of scoring a goal**.

- 0 → No chance  
- 1 → 100% goal  
- Example: 0.35 = 35% chance  

---

## ⚙️ How the Model Works

The formula used is:

xG = Zone Value × Pressure Factor × Balance Factor

Factors:
Near goal → High xG  |  Far from goal → Low xG
